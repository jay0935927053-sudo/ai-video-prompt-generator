# 🚀 部署到 GitHub Pages 指南

目標網址：https://jay0935927053-sudo.github.io/ai-video-prompt-generator/

---

## 第一步：安裝 Git

1. 前往 https://git-scm.com/download/mac
2. 下載並安裝 Git（按照預設設定即可）
3. 安裝後開啟「終端機」（Terminal），輸入以下指令確認安裝成功：
   ```
   git --version
   ```
   看到版本號就代表成功

---

## 第二步：安裝 Node.js（如果還沒有）

1. 前往 https://nodejs.org
2. 下載「LTS」版本並安裝
3. 安裝後確認：
   ```
   node --version
   ```

---

## 第三步：在 GitHub 建立新 Repo

1. 登入 https://github.com
2. 點擊右上角「+」→「New repository」
3. Repository name 填入：`ai-video-prompt-generator`
4. 選擇「Public」（GitHub Pages 免費版需要 Public）
5. **不要**勾選 Add README
6. 點擊「Create repository」

---

## 第四步：開啟終端機，進入專案資料夾

把這個 `ai-video-prompt-generator` 資料夾移到你方便的位置（例如桌面），然後：

```bash
cd ~/Desktop/ai-video-prompt-generator
```

---

## 第五步：安裝依賴套件

```bash
npm install
```

---

## 第六步：初始化 Git 並推送到 GitHub

依序執行以下指令（逐行複製貼上）：

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/jay0935927053-sudo/ai-video-prompt-generator.git
git push -u origin main
```

> 第一次 push 時，瀏覽器會彈出 GitHub 登入視窗，登入你的帳號授權即可。

---

## 第七步：開啟 GitHub Pages

1. 進入你的 repo 頁面：https://github.com/jay0935927053-sudo/ai-video-prompt-generator
2. 點擊上方「Settings」分頁
3. 左側選單找到「Pages」
4. 在「Source」下拉選單選擇「GitHub Actions」
5. 儲存

---

## 第八步：等待自動部署

- 回到 repo 頁面，點擊「Actions」分頁
- 你會看到一個正在執行的 workflow，等 1-2 分鐘
- 綠色勾勾代表部署成功 ✅

---

## 🎉 完成！

你的網頁網址：
**https://jay0935927053-sudo.github.io/ai-video-prompt-generator/**

## 日後更新網頁

每次修改 App.jsx 後，執行：

```bash
git add .
git commit -m "更新內容"
git push
```

GitHub Actions 會自動重新部署，約 1-2 分鐘後生效。
